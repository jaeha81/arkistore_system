# 02 MENU STRUCTURE

## JH 운영관리 대시보드 (ops-web)

```
[로그인]

[운영 대시보드 홈]
  ├─ 프로젝트 관리
  │   ├─ 프로젝트 목록/등록
  │   ├─ 프로젝트 상세
  │   └─ 사이트 등록
  ├─ 오류/이슈 관리
  │   ├─ 오류 신고 목록
  │   ├─ 이슈 그룹
  │   └─ GitHub 이슈 생성
  ├─ 이벤트 로그
  ├─ 오류 로그
  ├─ 배포 기록
  ├─ 운영 모드 관리
  ├─ 관리자 액션 로그
  ├─ 외부 연동 설정
  └─ 시스템 설정
```

## Arkistore 업무 시스템 (arki-web)

```
[로그인]

[메인 대시보드]
  └─ KPI 요약: 발주대기/재고부족/신규문의/오늘배송/CAPA/오류

[물류 사이트]
  ├─ 발주 요청
  │   ├─ 발주 요청 목록
  │   └─ 발주 요청 등록
  ├─ 발주서 관리
  │   ├─ 발주서 목록
  │   └─ 발주서 생성
  ├─ 인보이스
  │   ├─ 인보이스 목록
  │   └─ 인보이스 등록
  ├─ BL/선적 관리
  │   ├─ 선적 목록
  │   └─ BL 등록
  ├─ 재고 현황
  │   ├─ 재고 목록
  │   └─ 재고 수동 조정
  └─ 상품 마스터
      ├─ 상품 목록
      └─ 상품 등록

[영업 사이트]
  ├─ 고객 관리
  │   ├─ 고객 목록
  │   ├─ 고객 등록
  │   └─ VIP 관리
  └─ 문의/리드 관리
      ├─ 리드 목록
      └─ 리드 등록

[판매매니저 사이트]
  ├─ 상담 관리
  │   ├─ 상담 목록
  │   └─ 상담 등록
  ├─ 계약 관리
  │   ├─ 계약 목록
  │   ├─ 계약 생성
  │   └─ 첨부 관리
  ├─ 배송 관리
  │   ├─ 배송 목록
  │   └─ 배송 예약
  ├─ 배송 CAPA
  │   ├─ CAPA 현황
  │   └─ CAPA 등록
  └─ 해피콜
      ├─ 해피콜 목록
      └─ 해피콜 등록

[공지사항]
[도움말/FAQ]
[오류 신고]
[관리자 지원]
```
