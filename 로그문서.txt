방법 A — Docker Compose 한 방 실행 (권장)
  cd "C:\AI프로젝트\Arkistore Operations Automation System"
  docker compose up -d
  → DB + Redis + 마이그레이션 + API 자동 순서 기동

  cd apps\api
  python scripts/seed.py
  → 초기 계정 5개 + 샘플 제품 + CAPA 슬롯 30일치 생성

방법 B — 수동 실행
  □ docker run -d --name arkistore-db \
      -e POSTGRES_USER=arkistore \
      -e POSTGRES_PASSWORD=password \
      -e POSTGRES_DB=arkistore_db \
      -p 5432:5432 postgres:15-alpine
  □ pip install -e ".[dev]"  (apps/api 폴더에서)
  □ alembic upgrade head
  □ uvicorn app.main:app --reload --port 8000
  □ python scripts/seed.py  (초기 데이터)
  □ npm run dev (ops-web  → http://localhost:3000)
  □ npm run dev (arki-web → http://localhost:3001)

  http://localhost:8000/docs  → Swagger API 문서
